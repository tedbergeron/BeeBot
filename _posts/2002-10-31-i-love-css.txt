I love CSS. I found this site: link:http://www.scottandrew.com/[ScottAndrew.com]. What I really like is his left side boxes: the code goes like this: `div class="item"` followed by an `h2 class="itemhead"` and then in the style sheet:

[source, CSS]
----
.item {
   border-top:1px solid #cc0000;
   border-right:1px solid #cc0000;
   padding-right:10px;
   padding-top:10px;
   margin-bottom:40px;
   line-height:130%;
   font-size:80%;
}

.item form, .item p {
  margin-top:0px;
}

h2.itemhead {
  font-size:11px;
  display:inline;
  position:relative;
  top:-18px;
  background-color:#000;
  border:1px solid #cc0000;
  color:#ff9933;
  padding-left:4px;
  margin:0px;
  font-weight:normal;
  text-transform: uppercase;
}
----

absolutely cool!
